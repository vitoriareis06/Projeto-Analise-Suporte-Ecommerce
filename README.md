# Análise de Atendimentos de Suporte em E-commerce

## Contexto
Este projeto tem como objetivo analisar dados de atendimentos ao cliente de um e-commerce, buscando identificar os principais motivos de contato, os canais mais utilizados e oportunidades de melhoria no suporte.

## Dados
Os dados utilizados são públicos e foram obtidos na plataforma Kaggle, contendo registros de atendimentos de suporte ao cliente.

## Análises Realizadas
- Contagem de atendimentos por categoria de problema
- Contagem de atendimentos por canal de atendimento
- Análise cruzada entre categoria do problema e canal de atendimento

## Principais Insights
- A categoria **Returns** concentra o maior volume de atendimentos, indicando recorrência de solicitações relacionadas a devoluções.
- O canal **Inbound** é o mais utilizado pelos clientes, mostrando que o suporte é procurado ativamente em situações de problema.
- Problemas relacionados a **Returns** e **Order Related** concentram-se principalmente no canal Inbound.

## Ações Recomendadas
- Revisar e melhorar o processo de devolução.
- Investir em automação e materiais de autoatendimento.
- Criar fluxos específicos para categorias com maior volume de chamados.
